FUNDAMENTAL 1

Description

In this JavaScript course, you'll start by understanding why learning JavaScript is crucial for creating dynamic web applications. You'll write your first program, "Hello World," and learn how to link JavaScript to HTML for interactive websites. We will dive into values and variables, mastering conventions and types like strings, numbers, and booleans. You'll explore how to use let, const, and var for variable declarations, perform arithmetic operations, and understand operator precedence. You'll also learn to work with strings and template literals for cleaner, more efficient code. Decision-making will be covered with conditionals (if-else), and we'll explain the difference between type coercion and type conversion. We'll cover truthy and falsy values and use logical operators to enhance your decision-making code. Additionally, you'll understand how to use the switch statement for multiple conditions, differentiate between statements and expressions, and simplify logic using the ternary operator. Finally, you'll get an overview of JavaScript releases to understand the evolution of the language.

FUNDAMENTAL 2

Description

In this section of the JavaScript course, we'll dive into more fundamental topics, starting with Strict Mode, which helps catch potential errors in your code. You'll learn how to create reusable blocks of code with functions, and we'll compare function declarations vs. function expressions to understand their differences. We'll also cover the concise syntax of the arrow function and explore other general topics related to functions. You'll then learn about working with objects and arrays to manage collections of data effectively. The course will also introduce loops, starting with the for loop, followed by the while and do...while loops. Finally, you'll understand the use of break and continue keywords to control the flow of loops and exit early or skip iterations as needed.

DOM

Description

In this part of the JavaScript course, you'll be introduced to the DOM (Document Object Model), which allows JavaScript to interact with the HTML and CSS of a webpage. We'll start with an introduction to the DOM to cover the theory behind it. You'll learn how to select elements from the DOM using various methods and how to manipulate them to change content, styles, and attributes dynamically. Additionally, we'll cover how to use event handlers to respond to user interactions like clicks, key presses, and more, bringing your web pages to life.

ADVANCED

Description

In the advanced section of the course, we'll cover key advanced JavaScript concepts to take your skills to the next level. You'll learn about destructuring objects and arrays for easier extraction of values. ASYNCHRONOUS

Description

In this section, you'll learn about asynchronous JavaScript, which is essential for handling tasks like fetching data from a server. We'll start with an explanation of why async is important, covering the theory behind non-blocking code execution. You'll then dive into callbacks, which allow you to handle asynchronous operations. Next, we'll introduce promises to write cleaner and more manageable async code. You'll also learn how to handle errors in asynchronous operations using the try...catch block. Finally, we'll cover async/await, a modern syntax that makes working with promises even easier and more readable.

NODE.JS

Description

In this part of the course, we'll dive into Node.js, a powerful JavaScript runtime built on Chrome’s V8 engine. We'll begin by answering the question, what is Node.js?, covering the theory behind how it works. You'll learn about modules and how Node.js uses them to organize code. We'll discuss wrapper functions that Node.js uses to wrap your code. You'll also explore important native modules, starting with the path native module for working with file paths, the os native module for system information, and the file system native module for interacting with the filesystem. You'll also learn about the events native module and how the EventEmitter native module helps you work with events. We'll cover the HTTP native module for building web servers, and finally, you'll get familiar with NPM (Node Package Manager), which is essential for managing dependencies in your Node.js projects.

API

Description

The final part of the course will focus on the theoretical understanding of APIs (Application Programming Interfaces), which are crucial for enabling communication between different software systems. We'll begin by explaining what an API is, followed by a discussion of JSON (JavaScript Object Notation), the format commonly used for data exchange. You'll then learn about creating a server-side API using Axios, a promise-based HTTP client. Next, we'll cover the theory behind RESTful APIs, which follow specific architectural constraints. You'll also understand how API authentication works to secure data exchange. Finally, we'll touch on the process of making an API and integrating it with a database to make your web applications truly dynamic and data-driven.
